# Barber Services App

A cross-platform Flutter mobile app for barber services featuring user authentication, profile management, and service browsing.

## Features
- Email/password authentication with password reset
- User profile viewing and editing
- Service listings with detailed views
- Multi-screen navigation

## Tech Stack
- Flutter (Dart)
- Firebase Authentication
- Cloud Firestore

## Run Locally
```bash
git clone https://github.com/varunparekh123/flutter_barber_app.git
cd flutter_barber_app
flutter pub get
flutter run
